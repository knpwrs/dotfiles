# System Settings

This folder contains various configuration files for various system settings.

## macOS

Run [`macos.sh`] to set up custom settings. Read [`macos.sh`] for more info.

## Terminal.app

Open [`knpwrs.terminal`] to set up colors for Terminal.app.
